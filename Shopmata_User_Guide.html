1. Dashboard

The Dashboard is your central hub for monitoring store performance and accessing key metrics at a glance.

Overview

When you log in, you'll see the Dashboard displaying:

  • Today's Sales - Total revenue for the current day
  • Today's Buys - Total amount spent on buy transactions today
  • Pending Transactions - Number of buy transactions awaiting processing
  • Recent Activity - Latest actions taken in the system

Quick Actions

From the Dashboard, you can quickly:

  • Create a new buy transaction
  • Add a new product
  • Create a new order
  • View pending items

2. Customers

The Customers module helps you manage your customer database, track purchase history, and maintain relationships.

2.1 Managing Customers

Viewing All Customers

Navigate to Customers > All Customers to see your complete customer list.

The customer list displays:

  • Customer name
  • Email address
  • Phone number
  • Total transactions
  • Total spent/received
  • Last activity date

Adding a New Customer

  1. Click the "Add Customer" button
  2. Fill in the customer information:
    • First Name (required)
    • Last Name (required)
    • Email
    • Phone Number
    • Address
    • Lead Source - How they found you
  3. Click "Save"

Editing a Customer

  1. Click on the customer's name in the list
  2. Click "Edit"
  3. Update the necessary fields
  4. Click "Save Changes"

Customer Profile

Each customer profile shows:

  • Contact information
  • Transaction history (buys)
  • Order history (sales)
  • Memo history
  • Notes and communications
  • Tags assigned to the customer

2.2 Lead Sources

Lead Sources help you track how customers find your business, allowing you to measure marketing effectiveness and understand your best customer acquisition channels.

Managing Lead Sources

Navigate to Customers > Lead Sources to view and manage your lead sources.

Creating a Lead Source

  1. Click "Add Lead Source"
  2. Enter the source name (e.g., "Website", "Referral", "Walk-in", "Social Media")
  3. Add an optional description
  4. Click "Save"

Using Lead Sources

When creating or editing a customer, you can assign a lead source to track how they found you. This data helps you:

  • Understand which marketing channels are most effective
  • Allocate marketing budget to high-performing sources
  • Track customer acquisition trends over time

Common Lead Sources

  • Walk-in - Customers who visit your physical location
  • Website - Customers from your online presence
  • Referral - Customers referred by existing customers
  • Social Media - Customers from Facebook, Instagram, etc.
  • Google/Search - Customers who found you via search engines
  • Repeat Customer - Returning customers

3. Products

The Products module is where you manage your inventory, categorize items, and set up product templates for consistent data entry.

3.1 Managing Products

Viewing Products

Navigate to Products > All Products to see your product inventory.

Use the filters to narrow down your view:

  • Category - Filter by product category
  • Status - Published, Draft, or All
  • Stock Status - In stock, Out of stock, Low stock
  • Ring Size - For jewelry items
  • Metal Type - Filter by precious metal

Creating a New Product

  1. Click "Add Product"
  2. Fill in the basic information:
    • Title (required) - Product name
    • Description - Detailed product description
    • Category - Select the appropriate category
  3. Set up pricing:
    • Price - Selling price
    • Compare at Price - Original price for showing discounts
    • Cost - Your cost for margin calculations
    • Price Code - Internal pricing reference code
    • Charge Taxes - Whether to apply sales tax
  4. Add product images (drag and drop or click to upload)
  5. Fill in template attributes if a template is assigned
  6. Set inventory quantity and warehouse location
  7. Click "Save"

Product Variants

If your product comes in multiple variations (size, color, etc.):

  1. Toggle on "This product has variants"
  2. Add variant options:
    • Option 1 Name (e.g., "Size") and Value (e.g., "7")
    • Option 2 Name (e.g., "Metal") and Value (e.g., "Gold")
  3. Set price, SKU, and quantity for each variant

Product Images

Each product can have multiple images:

  • Primary Image - Main product photo (click the star icon)
  • Public Images - Visible on sales channels
  • Internal Images - For internal use only (certificates, receipts, etc.)

3.2 Categories

Categories help organize your products in a hierarchical structure.

Creating Categories

  1. Navigate to Products > Categories
  2. Click "Add Category"
  3. Enter the category name
  4. Select a parent category (or leave blank for a top-level category)
  5. Optionally assign a template
  6. Click "Save"

Category Hierarchy

Categories can be nested to any depth. For example:

  • Jewelry
    • Rings
      • Engagement Rings
      • Wedding Bands
    • Necklaces
    • Earrings

3.3 Product Types

Product Types are leaf categories (categories with no children) that have additional settings for SKU generation and barcode attributes.

Configuring Product Types

  1. Navigate to Products > Product Types
  2. Click on a product type to configure
  3. Set up:
    • SKU Prefix - Characters added before the SKU (e.g., "14K")
    • SKU Suffix - Characters added after the SKU
    • Default Bucket - Where items of this type are stored
    • Barcode Attributes - What information appears on labels
    • Charge Taxes - Default tax setting for this product type
  4. Click "Save Settings"

3.4 Templates

Templates define custom attributes for products, ensuring consistent data collection across similar items.

Creating a Template

  1. Navigate to Products > Templates
  2. Click "Create Template"
  3. Enter a template name and description
  4. Add fields by clicking "Add Field":
    • Field Name - Internal identifier
    • Label - What users see
    • Type - Text, Number, Select, Checkbox, etc.
    • Required - Whether the field must be filled
    • Options - For select/radio fields, add available choices
  5. Arrange fields by dragging to reorder
  6. Click "Save"

Assigning Templates to Categories

Templates can be assigned to categories, and all products in that category will inherit the template fields.

  1. Edit a category
  2. Select the template from the dropdown
  3. Save the category

4. Orders

The Orders module manages your sales, from creation through fulfillment.

4.1 Creating and Managing Orders

Creating a New Order

  1. Navigate to Orders > All Orders
  2. Click "Create Order"
  3. Select or create a customer
  4. Add products to the order:
    • Search for products by name, SKU, or barcode
    • Scan barcodes for quick entry
    • Adjust quantities as needed
  5. Apply any discounts
  6. Select shipping method (if applicable)
  7. Process payment
  8. Complete the order

Order Statuses

Status Description
Pending Order created, awaiting payment or processing
Processing Payment received, preparing for shipment
Shipped Order has been shipped
Delivered Order delivered to customer
Completed Order fulfilled and closed
Cancelled Order was cancelled

4.2 Shipments

Track and manage shipments for your orders.

Creating a Shipment

  1. Open an order
  2. Click "Create Shipment"
  3. Select items to ship
  4. Choose shipping carrier
  5. Enter tracking number
  6. Click "Ship"

4.3 Returns

Process customer returns and restocking.

  1. Navigate to Orders > Returns
  2. Click "Create Return"
  3. Select the original order
  4. Choose items being returned
  5. Select return reason
  6. Choose action: Refund, Exchange, or Store Credit
  7. Process the return

5. Buys (Transactions)

The Buys module manages purchasing transactions from customers - when customers sell items to you.

5.1 Creating a Buy Transaction

Starting a New Transaction

  1. Navigate to Buys > By Transaction
  2. Click "New Transaction"
  3. Follow the wizard steps:
    • Step 1: Customer - Search for existing customer or create new
    • Step 2: Items - Add items being purchased
    • Step 3: Payment - Enter offer and payment details

Adding Items

For each item the customer is selling:

  1. Click "Add Item"
  2. Select the category
  3. Enter item details:
    • Title/Description
    • Weight (DWT for precious metals)
    • Precious metal type
    • Condition
    • Estimated value
  4. Add photos of the item

Making an Offer

  1. Review all items
  2. System calculates suggested offer based on item values
  3. Adjust final offer if needed
  4. Select payment method:
    • Cash
    • Check
    • Store Credit
    • Wire Transfer
  5. Complete the transaction

5.2 Managing Buy Items

Navigate to Buys > All Items to see all purchased items across transactions.

Item Actions

  • Add to Inventory - Convert the item to a sellable product
  • Assign to Bucket - Organize items by storage location
  • Send to Repair - Items needing restoration
  • Send on Memo - Consign items to vendors

Viewing Item Details

Click on any item to view its detail page, which includes:

  • Item photos and description
  • Weight and material information
  • Estimated value and pricing
  • Transaction history
  • AI Research (see below)

5.3 AI-Powered Research

Each transaction item includes an AI Research feature that provides intelligent market analysis to help you make informed pricing decisions.

AI Research uses advanced AI to analyze your items based on their description, category, weight, and photos to provide market insights and pricing recommendations.

Using AI Research

  1. Open a transaction item's detail page
  2. Locate the "AI Research" card
  3. Click "Generate Research"
  4. Wait for the AI to analyze the item (usually a few seconds)
  5. Review the results

AI Research Results

The AI provides several types of insights:

Market Value Estimate

  • Low - Conservative market value estimate
  • Average - Typical market value
  • High - Premium market value estimate
  • Confidence Level - How confident the AI is in its estimate (High, Medium, Low)
  • Reasoning - Explanation of how the value was determined

Pricing Recommendations

  • Suggested Retail Price - Recommended selling price to end consumers
  • Suggested Wholesale Price - Recommended price for dealer/wholesale sales
  • Notes - Additional pricing considerations

Item Analysis

  • Description - AI-generated description of the item
  • Notable Features - Key characteristics that affect value
  • Condition Notes - Observations about the item's condition

Regenerating Research

If you've updated item details or want fresh analysis, click "Regenerate" to get updated AI research. This is useful when:

  • You've added more photos
  • You've updated the item description
  • Market conditions may have changed
  • You want a second opinion
Note: AI Research is a tool to assist your decision-making. Always use your professional judgment and market knowledge when making final pricing decisions.

5.4 Transaction Workflow

Status Description Next Steps
Pending Transaction created, awaiting review Review items, make offer
Items Reviewed Items have been evaluated Present offer to customer
Offer Accepted Customer accepted the offer Process payment
Payment Processed Payment has been made Add items to inventory
Offer Declined Customer declined the offer Return items or negotiate
Cancelled Transaction cancelled No action needed

7. Repairs

Track items sent for repair or restoration.

Creating a Repair

  1. Navigate to Repairs
  2. Click "New Repair"
  3. Select the item (from inventory or transaction)
  4. Choose the vendor/repair shop
  5. Enter repair details:
    • Description of work needed
    • Estimated cost
    • Expected completion date
  6. Click "Create"

Repair Workflow

Status Description
Pending Repair created, not yet sent
Sent Item sent to repair vendor
In Progress Repair work underway
Completed Repair finished, item returned

8. Memos

Manage consignment/memo arrangements with vendors.

Creating a Memo

  1. Navigate to Memos
  2. Click "New Memo"
  3. Select the vendor
  4. Add items to the memo
  5. Set memo terms:
    • Memo price for each item
    • Due date
    • Terms and conditions
  6. Click "Create Memo"

Memo Actions

  • Mark as Sold - Item sold by vendor, process payment
  • Return Item - Item returned unsold
  • Extend Memo - Extend the due date

Creating an Invoice

  4. Add line items
  5. Set payment terms
  6. Click "Save"

Invoice Actions

  • Send - Email invoice to customer

  6. Click "Save"

13. Reports

The Reports module provides comprehensive analytics and insights into your business performance. All reports can be exported to CSV for further analysis.

13.1 Sales Reports

Track your sales performance across different time periods.

Daily Sales Report

Navigate to Reports > Sales (Daily)

This report shows individual sales for a selected day, including:

Column Description
Order ID Unique identifier for the sale
Customer Customer name
Lead Source How the customer found you
Status Order status (Completed, Pending, etc.)
Marketplace Sales channel (In-store, Online, eBay, etc.)
# Items Number of items in the order
Categories Product categories sold
Cost Your cost for the items
Wholesale Value Wholesale price of items
Subtotal Sale amount before tax/shipping
Profit Net profit from the sale
Tax Sales tax collected
Shipping Shipping charges
Total Total amount collected
Payment Type How the customer paid
Vendor Associated vendor (if applicable)

Features:

  • Date picker to select any day
  • Summary totals at the bottom
  • Export to CSV button

Month over Month Report

Navigate to Reports > Sales (Month over Month)

Compare your sales performance across different months to identify trends and seasonal patterns.

Metrics Displayed:

  • Total sales per month
  • Number of orders
  • Average order value
  • Growth percentage vs. previous month
  • Visual charts for easy comparison

Month to Date Report

Navigate to Reports > Sales (Month to Date)

Track your progress for the current month against goals and previous periods.

Includes:

  • Current month sales total
  • Comparison to same period last month
  • Projected month-end total based on current pace
  • Daily breakdown chart

13.2 Buys Reports

Analyze your purchasing activity and profitability.

Online Buys Report

Navigate to Reports > Buys Report (Online)

Track transactions from online/mail-in sources.

Metric Description
Buys Count Number of online transactions
Purchase Amount Total amount paid to customers
Estimated Value Estimated resale value of items
Profit Projected profit (Estimated Value - Purchase Amount)
Profit % Profit as a percentage
Avg Buy Price Average transaction amount

Features:

  • Daily breakdown for the selected month
  • Visual charts showing trends
  • Week-over-week comparison
  • Monthly summary view available

In-Store Buys Report

Navigate to Reports > Buys Report (In Store)

Track walk-in customer transactions at your physical location.

Same metrics as Online Buys, filtered to in-store transactions only.

Trade-In Report

Navigate to Reports > Buys Report (Trade-In)

Track trade-in transactions where customers exchange items for store credit or other merchandise.

13.3 Inventory Report

Navigate to Reports > Inventory Report

Get a comprehensive view of your current inventory status.

Summary Statistics

At the top of the report, you'll see key metrics:

  • Total Stock - Total number of items in inventory
  • Total Value - Total retail value of inventory
  • Added This Week - Items added in the last 7 days
  • Cost Added - Cost of items added this week
  • Deleted This Week - Items removed/sold this week
  • Projected Profit - Expected profit if all items sell at listed price

Category Breakdown

The report shows inventory organized by category:

Column Description
Category Product category name (click to drill down)
Total Stock Number of items in this category
Total Value Retail value of items
Added This Week New items added
Cost Added Cost of new items
Deleted This Week Items removed/sold
Projected Profit Expected profit for this category

Weekly Trend Chart

A visual chart shows inventory movement over recent weeks:

  • Added - Items added to inventory
  • Removed - Items sold or removed
  • Net - Net change in inventory

Drilling Down

Click on any category to see its subcategories. Use the breadcrumb navigation to return to higher levels.

Time Period Options

The Inventory Report is available in multiple views:

  • Weekly - Last 7 days activity
  • Monthly - Last 30 days activity
  • Yearly - Last 12 months activity

13.4 Exporting Reports

All reports can be exported for further analysis:

  1. Navigate to the desired report
  2. Set your filters (date range, category, etc.)
  3. Click the "Export CSV" button
  4. Open the downloaded file in Excel or Google Sheets
Tip: Export reports regularly for record-keeping and to track long-term trends in spreadsheet software.
